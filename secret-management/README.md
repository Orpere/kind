# Secret Management

This directory contains documentation, configurations, and tooling for managing secrets across the infrastructure.

## Topics Covered

- Secret storage and rotation strategies
- Integration with external secret stores (Vault, AWS Secrets Manager, etc.)
- Kubernetes secret management (External Secrets Operator, CSI drivers)
- Encryption at rest and in transit
- Access control and auditing for secrets
- GitOps-compatible secret handling (SOPS, sealed secrets)
